● Use MongoDB to store the following data to get you started.




## Run Locally

Clone the project

```bash
  git clone https://github.com/sumit1617/Rest-Server.git
```

Go to the project directory

```bash
  cd server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon server.js
```
